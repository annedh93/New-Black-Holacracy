# Backend Developer 

## Purpose 

Toevoegen en onderhouden van de toegevoegde waarde van de EVA software 



## Accountabilities 

* Ontwikkelen van features in EVA back-end

* Implementeren van koppelingen met derde partijen

* Triageren en oplossen van software bugs

* Signaleren en aan @Undefined_Role communiceren van problemen met het behalen van de sprint

* Documenteren volgens de vastgestelde standaard van @Dr_Docs

* Faciliteren en onderhouden van documentatie en tooling

* Testen of use-case gewenste output geeft

* Properly documenting releases through release notes, in correlation with @Dr_Docs

* Making release notes accessible for the intended audience

* Facilitating guidelines (agreements, communication protocols, approval process, …)

* Facilitating tooling (GitHub, Jenkins, Henk, ...) for @Backend_Developer

* Onderzoeken van innovaties en nieuwe tools/technieken

* Toepassen/implementeren nieuwe tools/technieken

* Uitrollen van updates naar test- en productieomgevingen (i.o.m. @Klant_Manager )
